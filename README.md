A Java-based desktop application built using Swing, JDBC, and MySQL to manage local events and notify users efficiently. This project provides a simple and interactive interface for managing users, events, and notifications in a structured way.

🚀 Features
🔐 Login System (basic access control)
👤 User Management
Add users
View users (JTable)
Delete users
📅 Event Management
Add events
View events
🔔 Notification System
Notify users about events
View notifications
🔍 Search Functionality
Search users by name
📊 Dashboard
Menu-driven interface
🎨 GUI
Built using Java Swing
🛠️ Technologies Used
Java (JDK 8+)
Swing (GUI)
JDBC (Database Connectivity)
MySQL (Database)
IDE: Eclipse / VS Code
🗄️ Database Structure
users (id, name, email)
events (id, title, description, event_date, location)
notifications (id, user_id, event_id)
